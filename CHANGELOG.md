# 更新日志

## [1.0.0] - 2026-04-24

### 新增 ✨

#### 核心功能
- **房间管理**：完整的CRUD操作，支持批量编辑和删除
- **租约管理**：租期设置、租金管理、租客信息管理
- **收租记录**：租金收取记录、收租统计、欠租提醒功能
- **水电管理**：水表电表抄表、自动计费、费用统计、历史查询
- **自动通知**：水电录入后自动生成收租通知并发送到飞书群组
- **数据导出**：支持导出收租记录和水电数据为Excel格式
- **权限管理**：管理员和租客两种角色，权限控制完善

#### 技术特性
- **安全认证**：JWT令牌认证 + bcrypt密码加密
- **数据校验**：Pydantic v2数据验证 + SQLAlchemy 2.0 ORM
- **现代UI**：Vue 3 Composition API + TypeScript + Element Plus
- **性能优化**：前端懒加载、虚拟滚动、请求缓存
- **响应式设计**：支持桌面和移动设备

#### API端点
- 用户认证：注册、登录、获取当前用户信息
- 房间管理：列表、创建、更新、删除、批量操作
- 收租管理：创建、更新、删除、列表、统计
- 水电管理：抄表、历史查询、自动计费
- 数据导出：Excel格式导出
- 系统配置：默认费率设置

### 修复 🐛
