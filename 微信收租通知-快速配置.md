## 🔧 配置企业微信Webhook（可选）

### 步骤1：获取企业微信机器人Webhook

1. 打开企业微信群聊
2. 点击群设置 → 群机器人
3. 添加机器人 → 命名为"租房小助手"
4. 复制Webhook地址，格式类似：
   ```
   https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=693a91f6-7xxx-4xxx-9xxx-xxxxxxxxxx
   ```

### 步骤2：配置到系统

编辑配置文件：
```bash
cd ~/rent-management-system/backend
cp .env.wechat.example .env
```

编辑 `.env` 文件，填入你的webhook URL：
```bash
WECHAT_WEBHOOK_URL=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_KEY_HERE
```

### 步骤3：重启后端服务

```bash
# 如果服务正在运行，会自动重载（使用了 --reload 参数）
# 或者手动重启
cd ~/rent-management-system/backend
./venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

## 📱 使用方式

### 方式1：自动发送（普通房间）

1. 进入水电管理页面
2. 选择房间，录入水表读数
3. 录入同一天的电表读数
4. **系统自动发送微信通知** ✨

示例流程：
```
102房间
├─ 录入水表：358吨（2026-04-24）
├─ 录入电表：12551度（2026-04-24）
└─ ✅ 自动发送微信到租房群
```

### 方式2：手动催租（2501房间）

调用API：
```bash
curl -X POST "http://43.134.40.91:8000/api/reminders/send-rent-reminder/ROOM_ID" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**前端集成建议**：
在房间管理页面添加"发送催租通知"按钮：
```vue
<el-button
  v-if="room.room_number.startsWith('2501')"
  @click="sendRentReminder(room.id)"
  type="warning"
  icon="el-icon-message"
>
  发送催租通知
</el-button>
```

## ❓ 常见问题

### Q: 为什么没有收到微信消息？
A: 检查以下几点：
1. 是否配置了 `WECHAT_WEBHOOK_URL`
2. 水电是否在同一天录入
3. 房间是否有租客信息
4. 查看后端日志：`tail -f backend/logs/app.log`

### Q: 2501房间需要录入水电吗？
A: 不需要。2501房间的水电是分摊的，使用手动催租API即可。

### Q: 如何批量发送催租？
A: 可以遍历2501房间列表，调用催租API。或使用现有的 `/api/reminders/send-notifications` 接口。

### Q: 微信发送失败会影响数据保存吗？
A: 不会。微信发送是独立的异步任务，失败只会记录日志，不影响水电数据保存。
