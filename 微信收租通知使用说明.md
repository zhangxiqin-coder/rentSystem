# 微信收租通知功能使用说明

## 功能概述

系统现在支持**水电录入后自动发送微信收租通知**，分为两种场景：

### 1️⃣ 普通房间（有独立水电表）
- 录入水电数据后，**自动发送**完整的收租消息
- 消息包含：房号、租客、房租💰、水费💧、电费⚡、总计

### 2️⃣ 2501房间（水电分摊）
- 不需要录入水电数据
- 通过API**手动触发**催租通知
- 消息包含：房号、租客、房租💰（水电已分摊）

---

## 自动通知触发时机

**普通房间**：当你录入完**水和电两项数据**后，系统会自动发送微信通知。

示例：
```
1. 今天录入 102房 水表读数 → 不发送（等电表）
2. 今天录入 102房 电表读数 → ✅ 立即发送微信通知
```

---

## 新增API接口

### 为2501房间单独催租
```
POST /api/reminders/send-rent-reminder/{room_id}
```

**功能**：发送不含水电的催租通知

**适用场景**：
- 2501开头的房间（水电分摊）
- 其他只需催房租的场景

**请求示例**：
```bash
curl -X POST "http://43.134.40.91:8000/api/reminders/send-rent-reminder/1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**响应示例**：
```json
{
  "success": true,
  "message": "已发送 2501-1 催租通知",
  "room_number": "2501-1",
  "tenant_name": "张三",
  "rent_amount": 1500.00
}
```

## 前端集成建议

### 1. 水电录入页面
在录入水电数据的表单中，成功后可以提示用户：
```
✅ 水电数据已录入
📱 微信通知已发送给租客
```

### 2. 为2501房间添加"催租"按钮
在房间管理页面，为2501开头的房间添加一个按钮：
```vue
<el-button 
  v-if="room.room_number.startsWith('2501')"
  @click="sendRentReminder(room.id)"
  type="warning"
>
  发送催租通知
</el-button>
```

调用API：
```javascript
async sendRentReminder(roomId) {
  try {
    const res = await axios.post(
      `/api/reminders/send-rent-reminder/${roomId}`
    )
    this.$message.success(res.data.message)
  } catch (error) {
    this.$message.error('发送失败')
  }
}
```

## 注意事项

1. **微信发送失败不影响数据保存**
   - 系统会记录错误日志
   - 可以通过API重新发送

2. **2501房间不需要录入水电**
   - 避免在水电录入页面选择这些房间
   - 或在前端过滤掉2501开头的房间

3. **租客信息必须完整**
   - 房间必须有租客姓名
   - 否则无法生成消息
